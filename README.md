# Local Password Generator (C++)

This is a **basic C++ project** created to generate **random secure passwords locally**.

The main goal of this program is to provide a simple way to create strong passwords **without relying on external services, websites, or internet connections**, ensuring better privacy and security.

## Features

- Generates random passwords using:
  - Uppercase and lowercase letters
  - Numbers
  - Special characters
- Runs entirely **offline**
- Clears the terminal for a clean interface
- Works on **Windows and Linux**
- No passwords are stored permanently

## Why this project?

This project was created as a **learning exercise** and a **personal utility tool** to:
- Practice C++ fundamentals
- Work with random number generation
- Avoid using online password generators
- Keep password generation fully local and under user control

## Future Improvements

The next goal for this project is to add a **user-friendly interface** to improve usability.

Possible directions include:
- A simple **menu-based interface** in the terminal
- A **graphical user interface (GUI)** using a C++ framework
- User-defined password length and character sets

This will help turn the project from a basic utility into a more polished tool while continuing to run locally and offline.
